# Product Overview

One CLI for all of Google Workspace — built for humans and AI agents. It provides a dynamic command surface that adapts to Google's Discovery Service at runtime.

## Core Capabilities

- **Dynamic Command Surface**: Automatically builds CLI commands from Google's Discovery Service API schemas.
- **Agent-First Design**: Every response is structured JSON, optimized for consumption by AI agents.
- **Extensible Skills System**: Includes 100+ pre-built "Agent Skills" (SKILL.md) for common Workspace workflows.
- **Simplified Authentication**: Built-in helpers for OAuth2 setup, login, and credential management.

## Target Use Cases

- **Developer Productivity**: Managing Drive, Gmail, Calendar, and Sheets without manual `curl` calls.
- **AI Agent Tooling**: Providing a robust, structured interface for LLMs to interact with Google Workspace.
- **Workflow Automation**: Scripting repetitive tasks across multiple Google services with structured output.

## Value Proposition

`gws` eliminates the boilerplate of REST API integration by dynamically generating a CLI from Google's own API definitions. It bridges the gap between raw APIs and high-level agentic workflows with its built-in skills and structured JSON outputs.
